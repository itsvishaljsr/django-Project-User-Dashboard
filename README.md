
# 📝 My Django App

A **Django-based** web application that allows users to view and search contacts by category or name. The application includes user authentication features such as signup, login, and logout, and dynamically displays category-specific information.

## ✨ Features

- **🔒 User Authentication**: Signup, login, and logout functionality.
- **📂 Category Filtering**: Users can filter contacts by clicking on a category button.
- **🔍 Search Functionality**: Search contacts by name or category.
- **⚡ Dynamic Interface**: The homepage changes based on the user's login state.

## 🚀 Technologies Used

- **Django**: A high-level Python web framework that encourages rapid development and clean, pragmatic design.
- **Bootstrap**: A CSS framework for responsive and mobile-first front-end web development.
- **MySQL**: Used as the database to store application data.
